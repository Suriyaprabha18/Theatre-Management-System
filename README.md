# 🎬 Theatre Booking System

A modern Full Stack Movie Theatre Booking System that allows users to browse movies, select seats, make secure online payments, and manage bookings. The project also includes a powerful Admin Dashboard to manage movies, shows, users, and bookings.

## 🚀 Features

### 👤 User Module
- User Registration & Login (JWT Authentication)
- Browse Movies
- View Movie Details
- Seat Selection with Real-Time Availability
- Book Movie Tickets
- Secure Online Payment Integration (Razorpay)
- Booking Confirmation
- Booking History
- Responsive User Interface

### 🛠️ Admin Module
- Secure Admin Login
- Dashboard Overview
- Add, Update & Delete Movies
- Manage Shows
- View & Manage Users
- View Bookings
- Monitor Theatre Operations
